用於摩托車、汽車和重載卡車的液壓或離合器驅動的煞車系統。

保護煞車系統，避免受沉積和腐蝕傷害。

對金屬和橡膠件有良好的相容性。

黏度性穩定，不易受高溫影響。

超高乾沸點255℃，可防止高溫造成煞車失效。

適合搭載ABS防滑煞車系統用車。

具備優異的消泡性、抗氧化性。

超越美國運輸部DOT-4的規範。


油品規格符合：

CUNA NC 956-01 ; FMVSS 116 DOT 4 ; SAE J 1704 ; SAE J 1703 ; ISO 4925 Class 3 ; ISO 4925 Class 4


使用方法：

檢查更換煞車油請委託專業保養廠執行，以免造成安全問題。


注意事項：

選用適合之煞車油等級。

不同等級（號數）煞車油切勿混用。

煞車油經長期使用會因吸入水分導致沸點下降，雜質侵入使品質劣化，故請定期回廠保養更換。


主要成分：醇醚基礎油、抗腐蝕劑、橡膠保護劑等成分
